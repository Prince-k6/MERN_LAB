## Experiment 10

10. Develop a React code to collect data from rest API.

### Execution Steps:

1. **Verify Node & NPM Installation:**
   Check the installation in the terminal window by running:
   ```bash
   node -v
   npm -v
   ```

2. **Create the React App:**
   Create the React application by using `create-react-app` in the terminal:
   ```bash
   npx create-react-app rest-app
   ```

3. **Navigate to the Project Directory:**
   Change the directory by running:
   ```bash
   cd rest-app
   ```

4. **Add the React Code:**
   Open the file `src/App.js` in your editor and replace its contents with the code from `rest-app.js`.

5. **Start the Development Server:**
   In the terminal window, start the app:
   ```bash
   npm start
   ```

6. **Observe the Output:**
   Open the browser window at `http://localhost:3000` to see the user list (name and email) displayed dynamically from the REST API.
